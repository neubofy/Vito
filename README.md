# Veto

Veto is an advanced, proprietary anti-theft and tracking application for Android devices, accompanied by a Vercel-hosted Web Dashboard. It provides ultimate control over lost or stolen devices using Firebase and Cloudflare technologies.

## Features
- **Real-time Location:** Track your device accurately on a live map.
- **Advanced Theft Mode:** Captures photos, enables Bluetooth/Location, and plays max-volume alarms.
- **SMS Fallback:** Send SMS commands (`VETO LOCATE 1234`) to control your device without an internet connection.
- **Secure Web Dashboard:** Next.js based dashboard to send commands and view device status.
